0. Global Structure (ใช้ทุกหน้า)

Top Navigation Bar

  • Logo (ซ้าย)
  • Environment Status Light (Green / Yellow / Red)
  • Global Search (Agents / Orders / SKUs / Campaigns)
  • Notification Bell (Exception-only alerts)
  • Profile / Role Switcher

Left Sidebar (Collapsible)

Executive Overview, AI CEO, AI COO, AI CFO, AI CMO, AI Sales Director, AI R&D, AI Compliance, Health Integration, Logistics Control, Audit & Logs, System Settings

Layout Foundation

12-column responsive grid · Card-based modular design · Whitespace อย่างน้อย 24–32px

1. Executive Overview (Landing Page)

Purpose: Peripheral Awareness Dashboard

  • Section A: Strategic KPI Ribbon — Revenue, Inventory Risk Index, Cash Runway, Conversion Rate, Compliance Status
  • Status: Soft glow (Active) · Amber edge (Monitoring) · Red edge (Escalation)
  • Section B: Agent Activity Matrix (Grid 3x2) — Agent name, Confidence, Mission, Risk, Drill Down
  • Agents: CEO / COO / CFO / CMO / Sales / R&D / Compliance
  • Section C (Highlight): Exception Panel — low confidence, regulatory flags, cash flow warning, supply shortage

2. AI CEO — Strategic Command Center

  • Top: Macro Insight Panel (market trend, revenue projection, risk heatmap)
  • Middle: Scenario Simulator (budget slider, inventory risk toggle, product launch switch)
  • Output: forecasted revenue, margin shift, risk delta
  • Bottom: Strategic Recommendations Feed (Approve / Override / Simulate again)

3. AI COO — Supply Chain Control

  • Top: Demand Forecast Graph (SKU-level time-series, 30/60/90 day)
  • Middle: Inventory Distribution Map (warehouse nodes, stock heat gradient, cold-chain badge)
  • External Integrations: SCG JWD Logistics, Flash Express
  • Bottom: Risk Alerts (overstock, understock, delay probability)

4. AI CFO — Financial Intelligence

  • Top: Cash Flow Dashboard (real-time liquidity, AR aging, burn rate)
  • Middle: Invoice Processing Panel (OCR upload, auto-classified expenses, confidence score)
  • Bottom: Financial Risk Map (margin sensitivity heatmap, fraud anomaly alerts)

5. AI CMO — Hyper-Personalization Studio

  • Integrations: TikTok Shop, Google Cloud Natural Language
  • Top: Trend Radar (keyword clusters, sentiment polarity, viral probability)
  • Middle: Campaign Generator — inputs: segment, health profile cluster, budget; outputs: ad copy, video script, hooks
  • Buttons: Generate / A-B Test / Send to Compliance
  • Bottom: Performance Grid (ROAS, CTR, Engagement)

6. AI Sales Director — Conversational Commerce Console

  • Integration: LINE Messaging API
  • Left: Live Conversation Feed (threads, confidence badge, escalate to human)
  • Right: Recommendation Engine (suggested products, upsell bundle, margin preview)
  • Bottom: Checkout Pipeline (cart status, payment link, drop-off probability)

7. AI R&D — Formulation Lab

  • Top: Knowledge Graph Explorer (compound network, bioactive score)
  • Middle: Formula Builder (ingredient list, predicted solubility, taste simulation, bioavailability)
  • Right Sidebar: Ingredient Swapping Tool (supply risk %, cost delta, reformulation confidence)

8. AI Compliance — Regulatory Gate

  • Top: Content Review Queue (campaign drafts, labels, risk score)
  • Middle: RAG Evidence Panel (risky terms, regulatory snippets, confidence score)
  • Bottom: Decision Controls (Approve / Request Revision / Block + full audit log toggle)

9. Health Integration Screen

  • Integration: Terra API
  • Top: Consent Ledger (active users, data scope, expiry timeline)
  • Middle: Biomarker Dashboard (sleep score, HRV, activity index)
  • Bottom: Trigger Engine (if condition met → recommended campaign/product)

10. Logistics Control Center

  • Top: Order Flow Timeline (Ordered → Packed → Dispatched → Delivered)
  • Middle: Warehouse Grid (temperature zones, capacity %, dispatch queue)
  • Bottom: Delivery Verification (proof of delivery, face verification status, exception log)

11. Audit & Logs

  • Filters: Agent, Date, Confidence threshold, Decision type
  • Views: Prompt log, Version control, Regulatory archive snapshot, Export capability

12. System Settings

  • Agent confidence threshold slider
  • Human override policy
  • API key management
  • Model selection routing
  • Role-based access control

UX Core Principles Embedded

  • Peripheral awareness first
  • Exceptions only surface centrally
  • No real-time noise
  • Decision confidence always visible
  • Every AI action traceable
