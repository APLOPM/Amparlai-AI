Main Content Area (12-column responsive grid)

1. Executive Overview (Landing Page)

Purpose: Peripheral Awareness Dashboard

Section A: Strategic KPI Ribbon (Full Width)

Revenue (Today/MTD), Inventory Risk Index, Cash Runway, Conversion Rate, Compliance Status

Status indicators: Soft glow (Active) · Amber edge (Monitoring) · Red edge (Escalation)

Section B: Agent Activity Matrix (3x2)

Cards: Agent name, Confidence %, Current mission, Risk score, Drill Down

Agents: CEO / COO / CFO / CMO / Sales / R&D / Compliance

Section C: Exception Panel (Highlight)

  • Low confidence decisions
  • Regulatory flags
  • Cash flow warning
  • Supply shortage alert

2. AI CEO — Strategic Command Center

Top: Macro Insight Panel
Market trend graph · Revenue projection curve · Risk heatmap matrix
Middle: Scenario Simulator
Inputs: marketing budget slider, inventory risk toggle, product launch switch
Outputs: forecasted revenue, margin shift, risk delta
Bottom: Strategic Recommendations Feed
AI insights + Approve / Override / Simulate again

3. AI COO — Supply Chain Control

Top: Demand Forecast Graph
SKU time-series + 30/60/90 projections
Middle: Inventory Distribution Map
Warehouse nodes, stock heat gradient, cold-chain badge
Integrations: SCG JWD Logistics, Flash Express
Bottom: Risk Alerts
Overstock, understock, delay probability

4. AI CFO — Financial Intelligence

Top: Cash Flow Dashboard
Liquidity bar, AR aging, burn rate
Middle: Invoice Processing Panel
OCR upload, auto-classified expenses, confidence score
Bottom: Financial Risk Map
Margin sensitivity heatmap, fraud anomaly alerts

5. AI CMO — Hyper-Personalization Studio

Integrations: TikTok Shop, Google Cloud Natural Language

Top: Trend Radar
Keyword clusters, sentiment polarity, viral probability
Middle: Campaign Generator
Inputs: target segment, health profile cluster, budget
Outputs: ad copy, video script, hooks
Buttons: Generate / A-B Test / Send to Compliance
Bottom: Performance Grid
ROAS, CTR, engagement score

6. AI Sales Director — Conversational Commerce Console

Integration: LINE Messaging API

Left: Live Conversation Feed
Customer threads, AI confidence badge, escalate to human
Right: Recommendation Engine
Suggested products, upsell bundles, margin impact preview
Bottom: Checkout Pipeline
Cart, payment link, drop-off probability

7. AI R&D — Formulation Lab

Top: Knowledge Graph Explorer
Compound network visualization, bioactive score
Middle: Formula Builder
Ingredients, predicted solubility, taste simulation, bioavailability index
Right Sidebar: Ingredient Swapping Tool
Supply risk %, cost delta, reformulation confidence

8. AI Compliance — Regulatory Gate

Top: Content Review Queue
Campaign drafts, product labels, risk score
Middle: RAG Evidence Panel
Risky terms, regulatory snippets, confidence score
Bottom: Decision Controls
Approve / Request Revision / Block + Full audit log toggle

9. Health Integration Screen

Integration: Terra API

Top: Consent Ledger
Active users, data scope, expiry timeline
Middle: Biomarker Dashboard
Sleep score, HRV, activity index
Bottom: Trigger Engine
If condition met → recommended campaign/product

10. Logistics Control Center

Top: Order Flow Timeline
Ordered → Packed → Dispatched → Delivered
Middle: Warehouse Grid
Temperature zones, capacity %, dispatch queue
Bottom: Delivery Verification
POD, face verification status, exception log

11. Audit & Logs

Filters: Agent, Date, Confidence threshold, Decision type
Views: Prompt log, Version control, Regulatory archive snapshot, Export capability

12. System Settings

Agent confidence threshold slider
Human override policy
API key management
Model selection routing + RBAC

UX Core Principles Embedded

  • Peripheral awareness first
  • Exceptions only surface centrally
  • No real-time noise
  • Decision confidence always visible
  • Every AI action traceable
